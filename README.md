# 2. 使い方

Raspberry Pi Pico を使ったプログラムを置いています。各プロジェクトにて詳細確認ください。

- ST7735 : TFT ST7735 を使ったプロジェクト
- ST7789withTouch : タッチパネル付きの TFT ST7789 を使ったプロジェクト
